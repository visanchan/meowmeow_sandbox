# UI Kit · Receipt Admin

Desktop receipt browser. Same shape as the legacy `receipt_admin/` kit
(Header / StatStrip / ReceiptTable / ReceiptDetail), now in Mochi POS indigo,
with two SaaS additions:

- **Send Later** column + receipt badge
- **QR-on-receipt** block in the detail card (so the seller can see exactly
  what the customer scans to register their pet later)

Tokens come from `../../colors_and_type.css`.
