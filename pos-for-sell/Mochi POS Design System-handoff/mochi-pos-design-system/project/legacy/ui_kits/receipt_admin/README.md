# Receipt Admin — UI Kit

Recreation of `meowmeow_receipt_admin.html` — the post-event admin where staff browse sales, search receipts, and reprint slips.

**Canvas:** desktop, max-width 1200px, centered.

## Files
- `index.html` — list view + filter + receipt detail panel
- `Header.jsx` — wordmark + day filter + export
- `StatStrip.jsx` — today's totals (sales, items, avg)
- `ReceiptTable.jsx` — sales table
- `ReceiptDetail.jsx` — slip preview + actions
