# QR-menu empty state polish — 2026-05-22 14:49

## Business meaning
The customer-facing QR self-order menu (`/qr-menu`) is the **first thing a booth customer sees** when they scan the QR code. Its "booth not ready" state was bare centered text; now it gets the same illustrated empty treatment as the rest of the app — a friendly storefront icon in a soft lavender circle + a heavier title — so even the not-yet-set-up state looks intentional and on-brand to customers, not broken.

Visual/UI-only.
